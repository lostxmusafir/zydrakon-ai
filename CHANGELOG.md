# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-07-11

### Added
- Migrated codebase to Astro static site engine with Tailwind CSS v4.
- Created Home, About, Founder, Products, Research, and Contact pages.
- Created `MASTER_PLAN.md` detailing the company vision, brand identity, and design system.
- Created `ROADMAP.md` outlining the 10-phase long-term release cycle.
- Added AI-crawler optimizations (`llms.txt` and `robots.txt`).
- Configured dynamic JSON-LD structured schemas (`Organization`, `Person`, `WebSite`, `WebPage`, `ItemList`).
- Deployed website to production environment on Vercel.
