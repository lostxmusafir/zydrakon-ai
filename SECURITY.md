# Security Policy

## Supported Versions

Only the latest release version on the `main` branch is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 0.0.1| :white_check_mark: |

## Reporting a Vulnerability

We take the security of Zydrakon AI seriously. If you find a security vulnerability, please do not report it via public issues. Instead, email us directly at:

**security@zydrakon.ai**

Please include detailed steps to reproduce the vulnerability, along with any proof-of-concept scripts. We aim to acknowledge your report within 48 hours and provide a fix or mitigation path as quickly as possible.
